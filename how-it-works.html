How SourceLink Works

A dll and pdb are created by a compiler. SourceLink is used to create a text file (specification) that links source files to their raw download URLs. The pdbstr.exe is used to save the file to a stream in the pdb. The NuGet package for SourceLink.exe includes pdbstr.exe. Windows debuggers use srcsrv.dll to download the source files on demand. An md5 checksum stored in the pdb by the compiler must match the md5 checksum of the file that is downloaded.
